INSTALL
=======

```shell
git clone git@github.com:hft-lab/py-timex.git
# also may be cloned via https
# git clone https://github.com/hft-lab/py-timex.git
pip install -e py-timex
```

UPGRADE
=======
```shell
# to update to latest version
cd py-timex
git pull
```
